Dejar de usar var, usar siempre LET. CONS es para constantes nomás, es decir no se va a modificar a lo largo del código. Por ejemplo el número PI es siempre igual. 

Utilizar arrow function. Por ejemplo:

const funcion = () => console.log("hola").

Es lo mismo que: function funcion (){
console.log("hola")
}

CLASES: Las clases son similares a las funciones constructoras de objetos. Nos construyen un molde.

class animal {
    hablar (){
        return this.hablar()
    }

}
